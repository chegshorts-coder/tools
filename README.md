# Cheg's Barber Picks

A simple mobile-first affiliate landing page for Cheg.

## Edit Products

Update `src/products.js`:

```js
{
  name: "Budget Foil Shaver",
  image: "/images/budget-foil.jpg",
  link: "https://amzn.to/3QaqiTz"
}
```

Add real product images to `public/images/` using the matching filenames when ready. Missing images automatically show a clean placeholder.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
